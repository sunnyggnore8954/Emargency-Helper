##Emargency_Helper

이 프로젝트는 Flutter와 Dart를 기반으로 개발한 응급 환자 지원 모바일 애플리케이션입니다. GPS 기반 인근 응급실 탐색, 증상 입력형 자가진단과 상황별 응급 대처 매뉴얼, 캘린더 기반 복약 기록 및 알림, 의료 특화 AI 챗봇 기능을 하나의 서비스 흐름으로 구성하였으며, 사용자·응급 기록·복약 일정·챗봇 데이터를 연계하는 ERD 구조로 설계하였습니다. 사용자가 응급 상황에서 가까운 병원을 확인하고 필요한 대처 방법과 복약 정보를 신속하게 제공받을 수 있어, 환자의 골든타임 확보와 일상적인 건강 관리 지원에 활용할 수 있도록 설계되었습니다.
